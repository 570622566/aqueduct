aqueduct library

The core libraries of the Aqueduct framework.

Exports the following libraries:

  • 'logging'
  • 'safe_config'
  • 'aqueduct/application'
  • 'aqueduct/auth'
  • 'aqueduct/db'
  • 'aqueduct/http'
  • 'aqueduct/openapi'

See documentation guides at https://aqueduct.io/docs/.
